Project Wonderland: Toolkit for Building 3D Virtual Worlds
Experimental Technology sponsored by this community and Sun Microsystems Laboratories

Welcome. This is the home for Project Wonderland version 0.5. This site hosts our latest software and documentation. Project Wonderland is a 100% Java open source toolkit for creating collaborative 3D virtual worlds. Within those worlds, users can communicate with high-fidelity, immersive audio, share live desktop applications and documents and conduct real business. Wonderland is completely extensible; developers and graphic artists can extend its functionality to create entirely new worlds and add new features to existing worlds.

If you are looking for the existing Project Wonderland v0.4, please visit our original site at:
http://lg3d-wonderland.dev.java.net

About Version 0.5 Developer Release 1

This release provides developers with early access to a stable version of the v0.5 software so they can start porting existing v0.4 code or models developed for v0.4. This release has very limited functionality and in many respects does not resemble a virtual world. It is currently only available as source code (binaries are not yet available). It is not meant for general use.

Specifically, we expect developers to begin to:

  • Develop or port custom cell types and extensions to Project Wonderland using the new module system
  • Test importing models they have developed in the COLLADA format

In this developer release multiple clients can connect to a single server via Java Web Start. Avatars appear simply as teapots. Audio communication between avatars is supported. Shared X11 applications are not currently supported.

This release is the first of several "developer" releases, with each release adding new functionality towards a "user" release.
