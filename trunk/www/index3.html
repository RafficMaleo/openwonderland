Project Wonderland: Toolkit for Building 3D Virtual Worlds
Experimental Technology sponsored by this community and Sun Microsystems Laboratories

Welcome. This is the home for Project Wonderland version 0.5. This site hosts our updated software and documentation. Project Wonderland is a 100% Java and open source toolkit for creating collaborative 3D virtual worlds. Within those worlds, users can communicate with high-fidelity, immersive audio, share live desktop applications and documents and conduct real business. Wonderland is completely extensible; developers and graphic artists can extend its functionality to create entire new worlds and new features in existing worlds.

If you are looking for the existing Project Wonderland v0.4, please visit our original site at: http://lg3d-wonderland.dev.java.net

About Version 0.5 Developer Release 1

The purpose of this release is to allow developers the earliest possible access to a stable codebase of the v0.5 software, so that they may begin porting their existing code geared for the v0.4 codebase or models developed for v0.4. This release has very limited functionality and in many respects does not resemble a virtual world. It is currently only available as source code (binaries are not yet available). It is not meant for general use.

Specifically, we expect developers to begin to:

  • Develop or port custom cell types and extensions to Project Wonderland using the new module system
  • Test importing models they have developed in the COLLADA format

Currently, multiple clients may connect to a single server, but each client must run from the source code installed locally. Avatars appear simply at teapots, although audio communication between avatars is supported. Shared X11 applications are not currently supported.

This release represents the first of several "developer" releases that each will add on new functionality towards a "user" release.
