wonderlandtest_readme.txt


Dependencies:
    jmonkey package.



WonderlandTest project contains a port of the rtg animation code from the
previous version of Wonderland.


You can use (w, a, s, d) to navigate the camera forward, backword, left, and right
respectively.  You can move the mouse to rotate the camera.  Pressing 't' toggles
between wireframe and solid rendering.


class WonderlandTest.test.Main_Chris.java contains the main() function that when
setup to be the entry main() function, instantiates multiple animation instances.
when run, you see a line of many distint people animating.
